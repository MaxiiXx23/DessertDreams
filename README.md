### Projeto independente (Público) - 2023
### ***Dessert Dreams***
#### Sobre o projeto:
 * Site para uma confeitaria fictícia com Web Design feito por mim. Site desenvolvido utilizando o ecossistema React.js(Vite) com Typescript.
